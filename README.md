Pet Havens is a full-stack web application developed as part of a Computer Science course. The project simulates an online pet supply store where users can create accounts, log in, manage a shopping cart, and place orders.

## Features
- User sign-up and login
- Account management (password updates)
- Shopping cart with quantity updates
- Order placement and order history
- Database persistence using MySQL

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: PHP  
Database: MySQL  
Server Environment: MAMP  
Tools: VS Code, phpMyAdmin

## Local Setup Instructions
1. Install MAMP
2. Place the project folder inside the MAMP `htdocs` directory
3. Import the provided SQL file into phpMyAdmin
4. Start Apache and MySQL in MAMP
5. Open the project in a browser at `http://localhost`
